Event Management Application:


Event Registration


This is a feature of my Event Management App for who want to register event. In this page you can register your event in our app. You can't register the event with name, that already taken by any other Host or you. To register you have to fill up the registration form which you can see after click on "Event Registration". As you register any Event you got mail as acknowledgement of registration


This is a feature of my Event Management App for who want to participate in any event. In this page you can participate in any registered event whose deadline is not gone yet. For that you have to fill-up form for that click on "Participant Registration".As you register in participate you got sms as acknowledgement of registration. Make sure that you can only register in one event once

Event-Dashboard


This is a feature of my Event Management App for Host of event which are currently not finished. Host can see The details of Participant whi participated in Host's Event by entering valid Event Id and Password which was provided through Mail.


Services:

Email-Service:
This application has Email facility for acknowledgement of your registration that you have been registered this event. As you submit the form you got one Email that your details registered successfully.




SMS Service:
This application has SMS facility for acknowledgement of your registration that you have been registered in this event. As you submit the form you got one SMS on number that you filled in form that you registered your name in this event successfully. And you can find all the details of event in SMS.
